Assumption for this Program:
1- The item in the cart in considered as a book,food and medicine if the cart item String contains the substring present in the books,food and medicine array respectively.
2- The item in the cart is considered as imported if the "imported" keyword is present in the item string.

Instructions to run the program:
1-Install Node.js.
2-Install Typescript. Run the following command in the terminal :
npm install -g typescript
3-Compile the typescript file. Run the following command in the terminal :
tsc filename.ts
4-Run the js file created. Run the following command in the terminal :
node filename.js
